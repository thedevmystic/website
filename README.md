# About the Project

> A personal website where Math meets Programming and Philosophy meets Logic!

## Introduction

`thedevmystic.github.io` is a personal website created by me to showcase some of my talents.
Though I'm a Systems, OS, and Game Dev, I always have an interest in Web Development. So, I
thought to test it, I try to make this website on par with a industry-level professional
website so it's not your avergae day broken sites!

## Features

- **Beautiful**: Yeah, It's beautiful, atleast for me!
- **Modular**: Modular componenets for easy scaling.
- **Modern Stack**: Uses React, TailwindCSSv4, and other modern frameworks.
